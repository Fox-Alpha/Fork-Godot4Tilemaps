>DONE: Globals -> BuildingManager
	Tastendruck [1-5]
		-Emit Signal BUILDMODE_REQUESTED(Building Dictionary)

> Placer Node
	Event: - Receive Signal BUILDMODE_REQUESTED
		OnBuildModeRequested (BuildingID | Building IID | Building Dictionary)
		Load Building Image
		- Emit Signal BUILD_MODE_STARTED(ImageTexture2D_IID)	# Instance ID of loaded Image Resource
	Event: InputMouseMove:
		SetImageToMouseCursorPosition()
		SetGlobalMousePosition / UpdateGlobalMousePosition

> Building Layer
	Event: - Receive Signal BUILD_MODE_STARTED
	AWAIT MouseLeftClick()

> Navigation Layer
	Event: - Receive Signal BUILD_MODE_STARTED
		Calculate Buildingsize to TileMap
		Check Building Size in Layer, if can (NOT) placed
		{
			ImageWidth / TileSize
			ImageHeight / TileSize
			Calculate TileCount
		}
		-Emit Signal BUILD_MODE_CAN_PLACED(True|False)

> BuildingManager
	Event: - Receive Signal BUILDMODE_REQUESTED
		Preparing Building Instance
	Event: - Receive Signal BUILD_MODE_STARTED
		SetGlobalBuildMode = True
	AWAIT MouseLeftClick()

> Building Layer
	Event: - Receive Signal BUILD_MODE_STARTED
	AWAIT MouseLeftClick()
	Instantiate the Building on Coordinates
	Set as Child
	-Emit Signal BUILD_MODE_LEAVED()
